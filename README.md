# 終律世界

終律世界は、群像劇・マーダーミステリー・TRPG・小説制作を目的としたオリジナル世界設定プロジェクトである。

## コンセプト

- 魔法は奇跡ではなく工学である
- 魔力は世界に循環する有限資源である
- 魔法は「律式を綴り、媒体に魔法式を描き、魔力を消費して現象を実行する技術」である

## 概要

本プロジェクトでは以下を体系化する：

- 世界観設定
- 魔法体系（律式・魔法式）
- 種族
- 武術体系
- 魔導工学
- 国家・組織
- シナリオ群（マダミス・群像劇）

---

## 魔法体系（概要）

魔法は以下の手順で成立する：

1. 律式を綴る
2. 魔法式を構築する
3. 媒体へ描く
4. 魔力を供給する
5. 現象が発生する

---

## 律式

世界で唯一の術式言語。
すべての魔法は律式によって記述される。

基本律字は48文字で構成される。
